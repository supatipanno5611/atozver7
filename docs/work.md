# 작업 문서와 보관 문서

## 역할

설정된 작업 문서와 보관 문서를 열고, 작업 내용을 보관 문서에 백업한 뒤 작업 문서를 비웁니다.

## 제공 기능

- `작업 문서 열기`: 설정된 작업 문서를 메인 작업 영역에서 엽니다.
- `보관 문서 열기`: 설정된 보관 문서를 열고 마지막 heading으로 이동합니다.
- `모든 탭 닫기`: 메인 작업 영역의 고정되지 않은 탭을 닫습니다.
- `작업 문서 정리`: 작업 문서 내용을 타임스탬프와 함께 보관 문서 끝에 추가한 뒤 작업 문서를 비웁니다.
- 리본 아이콘으로 작업 문서와 보관 문서를 바로 열 수 있습니다.

## 설정 및 제약

- 작업 문서 경로, 보관 문서 경로, 백업 타임스탬프 형식은 설정값을 사용합니다.
- 대상 파일이 없으면 해당 작업을 수행하지 않습니다.
- 보관 문서의 마지막 heading 이동은 현재 에디터 텍스트에서 마지막 Markdown heading을 찾습니다.
