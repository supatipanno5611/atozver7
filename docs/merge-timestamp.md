# 타임스탬프 행 병합

## 역할

여러 행으로 선택된 타임스탬프 메모를 하나의 행으로 합칩니다.

## 사용 방법

- 합칠 행들을 선택합니다.
- 명령 팔레트에서 `타임스탬프 행 병합`을 실행합니다.
- 여러 선택 영역이 있으면 각 영역을 각각 병합합니다.

## 동작 예

```text
▶ 0:10 첫 문장
▶ 0:12 두 번째 문장
```

위 두 행을 선택해 실행하면 다음과 같이 바뀝니다.

```text
▶ 0:10 첫 문장 두 번째 문장
```

## 제약

- 타임스탬프 형식은 `▶ m:ss` 또는 `▶ h:mm:ss` 형태입니다.
- 한 행만 선택된 영역은 변경하지 않습니다.
- 첫 행에 타임스탬프가 없으면 합친 텍스트만 남깁니다.
