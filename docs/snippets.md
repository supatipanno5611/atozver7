# 조각글 제안

## 역할

에디터에서 설정된 조각글 트리거를 입력하면 저장된 문구를 검색해 삽입하거나 새 문구를 바로 등록합니다.

## 사용 방법

- 설정된 `snippetTrigger` 뒤에 검색어를 입력합니다.
- 목록에서 기존 조각글을 선택하면 트리거 입력 범위가 해당 문구로 바뀝니다.
- 일치하는 기존 조각글이 없는 입력은 `새 조각글로 추가` 항목을 선택해 삽입과 등록을 한 번에 수행할 수 있습니다.

## 설정 및 동작

- 표시 개수는 `snippetLimit` 설정을 따릅니다.
- 최근 사용한 조각글은 검색 점수에 작은 가중치가 더해집니다.
- 사용 기록과 새로 추가한 조각글은 설정에 저장됩니다.
