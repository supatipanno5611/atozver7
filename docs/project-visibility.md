# 프로젝트 폴더 숨김 토글

## 역할

설정된 프로젝트 폴더와 그 하위 폴더를 Obsidian 파일 탐색기에서 숨기거나 다시 표시합니다.

## 사용 방법

- 명령 팔레트의 `프로젝트 폴더 숨김 토글` 또는 폴더 아이콘 리본을 사용합니다.
- 숨김 상태는 설정에 저장되므로 플러그인을 다시 불러와도 유지됩니다.

## 동작 및 제약

- 설정의 `projectPath`가 필요합니다.
- 숨김을 켜면 플러그인이 생성한 스타일 규칙으로 대상 경로의 폴더 항목을 감춥니다.
- 파일 자체를 이동하거나 삭제하지 않으며, 파일 탐색기의 표시만 변경합니다.
