# 주제어 편집

## 역할

현재 마크다운 파일의 `topics` frontmatter 값을 추가하거나 제거합니다.

## 사용 방법

- 명령 팔레트에서 `주제어 편집`을 실행합니다.
- 기존 문서에서 수집한 주제어를 검색해 선택하거나, 입력한 새 주제어를 추가합니다.
- `[선택됨]`으로 표시된 주제어를 고르면 현재 문서에서 제거됩니다.

## 동작 및 제약

- 현재 문서에 `topics`가 없으면 빈 배열을 먼저 만듭니다.
- 주제어는 정렬되어 저장됩니다.
- 게시 노트 설정과 달리 프로젝트 폴더 밖의 마크다운 파일에서도 사용할 수 있습니다.
