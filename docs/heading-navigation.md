# 헤딩 이동

## 역할

현재 문서의 이전 또는 다음 heading으로 커서를 이동합니다.

## 제공 기능

- `이전 heading으로 이동`: 현재 줄 위의 가장 가까운 heading으로 이동합니다.
- `다음 heading으로 이동`: 현재 줄 아래의 가장 가까운 heading으로 이동합니다.

## 동작 및 제약

- heading 위치는 Obsidian의 현재 파일 메타데이터 캐시를 사용합니다.
- 더 이전 또는 다음 heading이 없으면 각각 첫 번째 또는 마지막 heading에 머뭅니다.
- heading이 없는 문서에서는 아무 동작도 하지 않습니다.
