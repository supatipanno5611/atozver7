# 내용을 잘라내어 새 노트 만들기

## 역할

현재 문서의 선택된 행 또는 문서 전체를 새 마크다운 노트로 이동합니다.

## 사용 방법

- 명령 팔레트에서 `내용을 잘라내어 새 노트 만들기`를 실행합니다.
- 텍스트가 선택되어 있으면 선택에 걸친 행 전체를 새 노트의 내용으로 사용합니다.
- 선택 영역이 없으면 현재 문서 전체를 새 노트로 이동합니다.
- 입력창에서 새 노트의 파일명을 지정합니다.

## 동작 및 제약

- 새 노트는 vault 루트에 `파일명.md`로 생성됩니다.
- 파일명이 비었거나 사용할 수 없는 문자를 포함하거나 같은 이름의 노트가 있으면 생성하지 않습니다.
- 새 노트를 만든 뒤 원본에서는 이동한 행 또는 전체 내용을 제거합니다.
