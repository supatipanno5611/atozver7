# 날짜 속성

## 역할

현재 마크다운 파일의 `date` frontmatter 값을 오늘 날짜로 기록합니다.

## 사용 방법

- 명령 팔레트에서 `오늘 날짜 속성 삽입`을 실행하면 `date`가 없을 때만 오늘 날짜를 추가합니다.
- 명령 팔레트에서 `오늘 날짜로 갱신`을 실행하면 기존 값과 상관없이 오늘 날짜로 덮어씁니다.

## 동작 및 제약

- 날짜는 `YYYY-MM-DD` 형식으로 저장합니다.
- `오늘 날짜 속성 삽입`은 이미 `date`가 있으면 값을 바꾸지 않습니다.
- 두 명령어 모두 현재 마크다운 파일에서만 사용할 수 있습니다.
