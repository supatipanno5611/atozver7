# 현재 파일 이동

## 역할

현재 마크다운 파일을 vault 안의 다른 폴더로 이동합니다.

## 사용 방법

- 명령 팔레트에서 `현재 파일 이동`을 실행합니다.
- `여기로 이동`을 선택하면 현재 선택된 위치로 파일을 이동합니다.
- 폴더를 선택하면 해당 폴더 안으로 들어갑니다.
- `새 폴더 만들기`를 선택하면 현재 위치 아래에 폴더를 만들고 그 안으로 파일을 이동합니다.

## 동작 및 제약

- vault root에서 탐색을 시작합니다.
- 활성 마크다운 파일에서만 사용할 수 있습니다.
- 대상 위치에 같은 이름의 파일이 있으면 이동하지 않습니다.
- 같은 이름의 새 폴더가 이미 있으면 만들지 않습니다.
- 새 폴더 이름에는 `/`, `\`, `..`를 사용할 수 없습니다.
