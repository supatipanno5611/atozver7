# 복사와 잘라내기

## 역할

현재 문서 전체 또는 현재 선택 영역을 클립보드로 복사하거나 잘라냅니다.

## 제공 기능

- `문서 전체 복사`: 현재 문서 전체 내용을 클립보드에 복사합니다.
- `문서 전체 잘라내기`: 전체 내용을 복사한 뒤 문서를 비웁니다.
- `복사하기`: 선택 영역을 복사합니다. 선택 영역이 없으면 현재 행을 복사합니다.
- `잘라내기`: 선택 영역을 잘라냅니다. 선택 영역이 없으면 현재 행을 잘라냅니다.
- 파일 메뉴의 `Copy whole document`: 메뉴에서 선택한 마크다운 파일 전체를 복사합니다.

## 제약

- 클립보드 쓰기가 가능한 환경에서 동작합니다.
- 내용이 없는 문서를 전체 잘라내기 할 때는 아무 변경도 하지 않습니다.
