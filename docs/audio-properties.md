# 오디오 속성 삽입

## 역할

현재 파일의 frontmatter에 외부 오디오 링크와 표시 제목을 저장합니다.

## 사용 방법

- 명령 팔레트에서 `오디오 속성 삽입`을 실행합니다.
- 먼저 오디오 링크를 입력하고, 이어서 오디오 제목을 입력합니다.
- 저장되는 속성은 `audioSrc`와 `audioTitle`입니다.

## 제약

- 오디오 링크는 `http://` 또는 `https://` URL이어야 합니다.
- 활성 파일과 비어 있지 않은 제목이 필요합니다.
