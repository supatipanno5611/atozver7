# 게시 노트 설정

## 역할

프로젝트 폴더 안의 노트를 게시 형식에 맞게 설정하고 필요한 frontmatter를 기록합니다.

## 사용 방법

- 명령 팔레트에서 `게시 노트 설정`을 실행합니다.
- 일반 게시글은 `date`와 `topics`를 준비합니다.
- 일상 게시글은 노트를 `<projectPath>/ordinary/` 바로 아래로 이동한 뒤 일반 게시글처럼 설정합니다.
- 목차 문서는 `type: index`만 기록합니다.
- 시리즈 게시글은 목차 문서와 순번을 선택한 뒤 `date`, `topics`, `parent`, `order`를 기록합니다.
- 주제어 값 편집은 `주제어 편집` 명령어에서 따로 합니다.

## 동작 및 제약

- 설정의 `projectPath` 안에 있는 마크다운 노트에서만 사용할 수 있습니다.
- 이미 게시 속성이 있으면 `초기화` 또는 `취소`를 선택합니다.
- 초기화하면 게시 속성을 제거하고 노트를 `<projectPath>/` 바로 아래로 이동합니다.
- 일반, 일상, 시리즈 게시글로 다시 설정할 때 기존 `topics` 값은 유지합니다.
- 목차 문서로 설정할 때는 `date`, `topics`, `parent`, `order`를 제거합니다.
- `parent`는 프로젝트 경로 기준의 문서 경로이며 `.md` 확장자를 포함하지 않습니다.
- 시리즈 게시글은 같은 부모 아래에서 이미 사용 중인 `order`를 저장하지 않습니다.
