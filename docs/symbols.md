# 기호 제안과 스마트 삭제

## 역할

설정된 트리거로 기호를 검색해 삽입하고, 짝 기호 사이에서 Backspace를 눌렀을 때 두 기호를 함께 지웁니다.

## 사용 방법

- 설정된 `symbolTrigger` 뒤에 기호 ID 검색어를 입력합니다.
- 기호에 닫는 문자 설정이 있으면 선택 영역을 감싸거나, 선택이 없을 때 쌍을 삽입하고 커서를 가운데에 둡니다.
- 후보가 하나뿐이면 자동으로 삽입될 수 있습니다.

## 설정 및 동작

- 표시 개수는 `symbolLimit`, 짝 삭제는 `symbolPairs` 설정을 따릅니다.
- 최근 사용한 기호는 검색 순위에 반영되어 저장됩니다.
